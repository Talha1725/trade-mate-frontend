# Trade Mate

Trade Mate is a Next.js and shadcn/ui project scaffolded with a feature-first
source structure.

## Project Structure Standard

- `src/app` contains routes, layouts, and API route handlers.
- `src/components` contains shared UI components.
- `src/components/ui` contains shadcn components and base UI primitives.
- `src/features/<feature-name>` contains feature-specific workflows.
- `src/hooks` contains shared React hooks.
- `src/lib` contains shared helpers and integrations.
- `src/types` contains shared TypeScript types.

## Current Setup

- Next.js App Router enabled
- `src/` directory enabled
- shadcn initialized
- `src/components/ui/button.tsx` installed
- `src/lib/utils.ts` installed
- shared types folder created at `src/types`

## Run Locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to see the starter page.
